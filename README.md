# Greys_Sports_Almanac

An experimental signal-analysis platform exploring how AI agents can monitor complex systems and surface useful patterns.

Inspired by the fictional sports almanac from Back to the Future — except instead of time travel, this project experiments with automated observers that continuously analyze signals from markets, data streams, and other complex systems.

## Current Architecture

- Claude CoWork running Claude Code via PowerShell
- Ubuntu Server (HP EliteBook G4) hosting PostgreSQL databases
- AI agents analyzing signals and storing observations
- Modular architecture designed for experimentation

## Status

Early lab project. Mostly curiosity-driven architecture experiments.
